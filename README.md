# Taifhum
Taifhum

Taifhum is a concept UI prototype designed for an Absher hackathon.
It presents a clear front-end flow that organizes services into three sections and shows how the idea would work in a real system.

Overview

The prototype focuses on a simple, user-friendly interface that routes users to the correct service type:

Absher Individuals

Absher Business (Schools and Nurseries)

Absher Services (Government Entities)

Key Sections
1) Absher Individuals

Personal services for citizens and residents.

2) Absher Business

A business portal targeted for:

Schools

Nurseries

Private sector entities (optional based on scope)

3) Absher Services

A portal for:

Government entities

Official authorities and service providers

Prototype Scope
Testing / Verification

Live testing and real verification were not implemented in this stage.
The current deliverable is a UI prototype that explains the concept, user flow, and intended functionality.

Features (UI)

Landing page with the 3 main sections

Clear navigation and user journey

Screens explaining “how it works”

Placeholder screens for future integration (API, verification, authentication)

Tech Stack

Frontend: (e.g., React / Next.js / Flutter / HTML-CSS-JS)

UI: (e.g., Tailwind / Material UI)

Prototype: UI-only (no backend integration)
